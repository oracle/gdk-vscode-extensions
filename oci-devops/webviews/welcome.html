Prerequisites

  • • Git - required for core functionality
  • • Java (JDK 17+)
  • • Kubectl - used to access applications deployed to OKE
  • • A Docker-API compatible container runtime such as Rancher Desktop or Docker

Setup

  1. 1. Create Oracle Cloud account (free tier supported)
  2. 2. Create .oci/config cloud access configuration file
  3. 3. Install OCI CLI needed for the OKE local access
  4. 4. Create and setup access to an OKE cluster. OKE is Oracle's managed Kubernetes service.
